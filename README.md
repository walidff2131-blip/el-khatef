# 🦅 الخاطف - El Khatef (Video Downloader)
**الخاطف** هو أداة جزائرية مفتوحة المصدر لتحميل الفيديوهات والصوت من مختلف منصات التواصل الاجتماعي (YouTube, TikTok, Instagram...) بجودة عالية.

## ✨ المميزات:
- 🇩🇿 واجهة بهوية جزائرية تقنية (Hacker Style).
- 🎬 تحميل الفيديوهات بجميع الجودات حتى 4K.
- 🎵 استخراج الصوت فقط بصيغة MP3.
- ⚡ سريع، خفيف، ولا يحتاج لسيرفر (Client-side).
- 📱 متوافق تماماً مع أجهزة الأندرويد والموبايل.

## 🛠️ التقنيات المستخدمة:
- HTML5 / CSS3 (Matrix Background Animation).
- Pure JavaScript (ES6+).
- API: [Cobalt](https://cobalt.tools).
